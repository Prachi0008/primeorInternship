About the Program

The Primeor Solutions Web Developer Internship is a 12-week program designed to give aspiring developers real hands-on experience. You will work alongside senior developers, contribute to live projects, and build a portfolio you can be proud of.

The program is split into three phases: onboarding (weeks 1–2), team project work (weeks 3–8), and a personal capstone project (weeks 9–12). By the end, you'll have practical skills, industry connections, and a certificate of completion.

Weeks 1–2 Onboarding & Tooling
Weeks 3–8 Team Project Work
Weeks 9–12 Capstone Project
Day 84 Demo Day & Certificate

Program Highlights

Here's what you'll gain from joining our internship:

Qualifications

We're looking for motivated learners. You don't need to know everything — just the basics and a drive to grow.

Required Skills

  • HTML5 & CSS3
  • Basic JavaScript (ES6+)
  • Git & GitHub
  • Responsive Design
  • Basic understanding of REST APIs

Nice to Have

  • React or Vue.js
  • TypeScript
  • Figma / UI tools
  • Node.js basics
  • Tailwind CSS

What We Look For

  • ✓ Enrolled in or recently graduated from CS, IT, or a related program
  • ✓ Available for the full 12-week duration
  • ✓ A GitHub profile or small personal project to share
  • ✓ Good communication skills
  • ✓ Eager to learn and take feedback positively
  • ✓ Ability to work independently and in a team
